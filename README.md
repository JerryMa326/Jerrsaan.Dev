# ChemClub Analyst

> A web-based image analysis tool for colorimetric assay detection and regression analysis.

## Overview

ChemClub Analyst helps chemistry labs detect, quantify, and analyze color samples in well plates with precision. Upload images of your assays, automatically detect wells, extract color data, and perform regression analysis—all in the browser.

## Key Features

- **Image Loading** — Drag-and-drop support for batch analysis
- **Auto Detection** — OpenCV.js-powered Hough Transform for circles, contour analysis for rectangles
- **Manual Adjustment** — Fine-tune detected regions for accuracy
- **Regression Analysis** — Chart.js visualizations for concentration curves
- **Data Export** — Export results for further analysis

## Tech Stack

React, TypeScript, OpenCV.js, Chart.js, Tailwind CSS
